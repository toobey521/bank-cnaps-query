# 🏦 银行网点行号查询工具

全国CNAPS联行号查询系统，支持按银行名称、地区、网点名模糊搜索。

## 功能

- 🔍 **模糊搜索**：支持银行名、地区名、网点名混合搜索
- 📍 **城市优先**：搜索结果按大城市优先排序
- 📋 **完整信息**：显示联行号、银行名称、网点地址、联系电话
- ⚡ **实时查询**：数据实时获取，无需刷新

## 快速开始

1. 打开页面
2. 输入银行名称或城市
3. 选择结果查看详细信息

## 技术栈

- 前端: 原生 HTML5 / CSS3 / JavaScript
- 部署: GitHub Pages
- 数据API对接
